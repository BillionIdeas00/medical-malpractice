# Medical Malpractice Guide Website

## Overview
This is a comprehensive, multi-page medical malpractice guide website designed to help individuals determine if they have a valid case. The site covers all types of medical malpractice cases with detailed information across 9 interconnected pages.

## Files Included
- `index.html` - Home page with overview and navigation to all sections
- `what-is-malpractice.html` - Defining medical malpractice and standards of care
- `types-of-cases.html` - Common types of malpractice cases (12+ categories)
- `proving-your-case.html` - Four essential elements needed to prove a case
- `statute-of-limitations.html` - Time limits and deadlines by state
- `finding-attorney.html` - How to find qualified legal representation
- `what-to-expect.html` - Step-by-step legal process timeline
- `damages.html` - Types of compensation available
- `faq.html` - Frequently asked questions
- `style.css` - Complete stylesheet for all pages

## How to Use with GitHub

### Option 1: GitHub Pages (Recommended)
1. Create a new repository on GitHub
2. Upload all files (HTML and CSS) to the repository
3. Go to Settings > Pages
4. Select the branch (usually "main") and root folder
5. Click Save
6. Your site will be live at: `https://yourusername.github.io/repository-name`

### Option 2: Direct Upload
1. Create a new repository
2. Click "Add file" > "Upload files"
3. Drag and drop all HTML and CSS files
4. Commit the changes
5. Enable GitHub Pages in Settings
