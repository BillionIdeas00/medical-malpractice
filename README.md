# Philadelphia Medical Malpractice Lawsuit Guide

A comprehensive, educational static website providing authoritative information about medical malpractice law in Philadelphia and Pennsylvania.

## Overview

This website serves as an educational resource for individuals seeking to understand medical malpractice law in Pennsylvania. It covers essential topics including:

- Elements required to prove medical malpractice
- Common types of medical negligence
- Pennsylvania's statute of limitations
- Compensation available to victims
- Frequently asked questions
- Legal procedures and requirements

## Purpose

This is an informational resource only and does not constitute legal advice. The site is designed to help families understand complex medical malpractice concepts and recognize when professional legal consultation may be warranted.

## Deployment to GitHub Pages

### Quick Setup

1. Create a new repository on GitHub
2. Upload `index.html` and `style.css` to the repository
3. Go to repository Settings → Pages
4. Under "Source", select "Deploy from a branch"
5. Select the `main` branch and `/ (root)` folder
6. Click Save
7. Your site will be published at `https://[username].github.io/[repository-name]/`

### File Structure

```
repository-root/
├── index.html
├── style.css
└── README.md
```

### Custom Domain (Optional)

To use a custom domain:

1. Add a `CNAME` file to the repository root with your domain name
2. Configure DNS settings with your domain registrar
3. Add your custom domain in repository Settings → Pages → Custom domain
